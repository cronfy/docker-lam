Перед тем как пробовать запустить сайт в первый раз:

1. Проверить, что в .htaccess нет редиректов на другое доменное имя, если есть - закомментировать

После того как сайт завелся:

1. Прописать доменное имя:
  1.1. Настройки продукта -- Сайты -- Список сайтов -- выбрать сайт
    1.1.1. Доменное имя
    1.1.2. URL сервера (без http://)
  1.2. Настройки продукта -- Настройки модулей -- Главный модуль
    1.1.1. Название сайта
    1.1.2. URL сайта (без http://, например www.mysite.com)
