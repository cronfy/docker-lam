Перед тем как пробовать запустить сайт в первый раз:

1. Проверить, что в .htaccess нет редиректов на другое доменное имя, если есть - закомментировать

После того как сайт завелся:

1. Прописать доменное имя:
  1.1. Настройки продукта -- Сайты -- Список сайтов -- выбрать сайт
    1.1.1. Доменное имя
    1.1.2. URL сервера (без http://)
  1.2. Настройки продукта -- Настройки модулей -- Главный модуль
    1.1.1. Название сайта
    1.1.2. URL сайта (без http://, например www.mysite.com)

2. Убедиться, что отправка писем по SMTP отключена (отключить модуль SMTP). Почта должна
отправляться только через PHP, иначе мы будем слать тестовые письма клиенту.

3. Убедиться, что сайт не обращается ко внешним сервисам НА ЗАПИСЬ в процессе работы. Например, 
при создании заказа не стучится в какую-нибудь облачную CRM и не создает там заказ. Иначе наши
тесты будут влиять на боевую систему заказчика.
Каким-либо образом это обнаружить нельзя, так что надо просто знать, что такое есть или спросить
у заказчика/прежнего разработчика.
